🚀 Deploy & Sync Setup

To enable real-time multi-device sync, connect a free Supabase project. It takes about 5 minutes.

1
Create a Supabase project
Go to supabase.com → New Project → any name
2
Get your API Keys
Project Settings → Add App → Web (</>) → copy the Project URL and anon key
3
Enable Firestore
Build → Firestore Database → Create database → Start in test mode
4
Enable Anonymous Auth
Build → Authentication → Get started → Sign-in method → Anonymous → Enable
5
Paste config into this file
Open the .html file → find SUPABASE_CONFIG → paste your values → set ENABLE_SUPABASE = true
6
Deploy (free options)
GitHub Pages · Netlify · Vercel · Supabase — all free, just drag & drop your .html file
📋 Recommended Firestore Security Rules 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Only authenticated (anonymous) users can read/write
    match /erp_{collection}/{docId} {
      allow read, write: if request.auth != null;
    }
    match /erp_presence/{staffId} {
      allow read, write: if request.auth != null;
    }
    match /erp_chat/{msgId} {
      allow read, write: if request.auth != null;
    }
  }
}
Paste these in Firestore → Rules tab and click Publish
